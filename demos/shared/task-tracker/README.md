# Task Tracker Sample Project

This sample is intentionally simple and is used by all demos.

## Functional Goal

A tiny CLI to add, list, and complete tasks in a local JSON file.

## Why This Project

1. Fast to understand in less than 5 minutes.
2. Small enough to focus on stack capabilities instead of app complexity.
3. Good fit to demonstrate AI-assisted iteration, reviews, and audits.

## Scope

1. No external services.
2. No database server.
3. Single local execution flow.
