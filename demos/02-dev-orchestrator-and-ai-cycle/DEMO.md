# Demo 02 - Orchestrator and AI Cycle

Audience: Development Team

## Goal

Show how the orchestrator and skills drive AI-assisted implementation flow.

## Scope

1. Session context generation.
2. Skill-aware workflow.
3. Custom rules injection.
4. Task execution with AI alignment.

## Components Demonstrated

1. `context-pack.ps1`
2. `orchestrator-next-steps.ps1`
3. `custom-rules.ps1`
4. Skills: project orchestrator + session workflow + technical skills

## Run Steps

1. `./scripts/utilities/wf.ps1 context-pack`
2. `./scripts/utilities/orchestrator-next-steps.ps1`
3. `./scripts/utilities/custom-rules.ps1 status`
4. `./scripts/utilities/custom-rules.ps1 export`

## Expected Outcome

1. AI receives consistent context and rule set.
2. Suggested actions are ordered by orchestration logic.
3. Team sees reduced manual coordination overhead.
