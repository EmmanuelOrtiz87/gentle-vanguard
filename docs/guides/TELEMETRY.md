# Telemetry Guide

## Purpose

Telemetry tracks estimated token usage and cost impact for workflow tasks.

## Data Sources

- `docs/sessions/metrics/token-guard-usage.csv`
- Generated by `scripts/utilities/token-budget-guard.ps1` when `-Record` is used.

## Reports

Generate a report:

```powershell
.\scripts\utilities\token-telemetry-report.ps1
```

JSON output:

```powershell
.\scripts\utilities\token-telemetry-report.ps1 -AsJson
```

## Cost Settings

Defaults:
- `CostPer1MTokens = 10`
- `BaselineTokensPerTask = 14000`
- `ReductionPercent = 40`

Adjust by passing parameters to the report script.
