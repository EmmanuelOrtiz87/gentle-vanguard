## When to Use

- Writing job descriptions and posting roles
- Screening and assessing candidates
- Designing interview processes and rubrics
- Managing offer negotiations and closures
- Creating onboarding plans for new hires

## 📋 Technical Deliverables

### Job Description Template

```
## Job Description: [Role Title]
**Department**: [Engineering/Product/Design]
**Level**: [Junior/Mid/Senior/Staff]
**Location**: [Remote/On-site/Hybrid]
**Salary Range**: [$X - $Y]

## About the Role
[2-3 sentences on what this person will do]

## Responsibilities
- Build and maintain [system/feature]
- Collaborate with [teams] to [outcome]
- Mentor junior team members

## Requirements
### Must-Have
- 3+ years [technology/domain]
- Experience with [key skill]
- [Soft skill] and communication

### Nice-to-Have
- [Certification/degree]
- Open source contributions
- [Secondary skill]

## Benefits
- Health, dental, vision
- 401k match
- [Other perks]
```

### Interview Rubric

```
## Interview Rubric: [Role]
**Interview Type**: Technical Screen
**Duration**: 45 minutes

## Evaluation Criteria
| Criteria | Weight | Score 1-5 | Notes |
|----------|--------|------------|-------|
| Coding Skills | 40% | | |
| System Design | 30% | | |
| Communication | 20% | | |
| Culture Fit | 10% | | |

## Scoring Guide
5 = Exceeds expectations (strong hire)
4 = Meets expectations (hire)
3 = Borderline (lean no-hire)
2 = Below expectations (no-hire)
1 = Significantly below (strong no-hire)

## Red Flags
- Cannot explain their past projects
- Dismissing feedback or questions
- [Role-specific red flags]
```

## 🔄 Workflow Process

### Step1: Role Definition & Sourcing

- Define role requirements with hiring manager
- Write inclusive job description (avoid bias words)
- Post to job boards (LinkedIn, Indeed, niche boards)
- Source passive candidates (LinkedIn, GitHub, Twitter)

### Step2: Screening & Assessment

- Review resumes against must-haves
- Phone screen top candidates (30 min)
- Send take-home assessment or schedule technical screen
- Score using standardized rubric

### Step3: Interview Process

- Coordinate interview panel (4-5 interviews)
- Ensure diverse interview panels
- Collect feedback within 24 hours
- Make decision in hiring committee

### Step4: Offer & Onboarding

- Extend offer with clear deadlines
- Negotiate salary/equity within band
