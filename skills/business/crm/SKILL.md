# CRM Skill

## When to Use

User asks about CRM platforms, customer data management, sales pipelines, lead scoring, customer
segmentation, or CRM integration.

## Core CRM Platforms

- **Salesforce**: Sales Cloud, Service Cloud, Marketing Cloud, Apex, SOQL, triggers
- **HubSpot**: Contact management, deal pipeline, marketing hub, workflows, custom properties
- **Zoho CRM**: Modules, workflows, blueprints, Zoho Deluge scripting
- **Microsoft Dynamics**: Sales, Customer Insights, Power Platform integration

## Response Patterns

1. Identify which CRM platform the user is referring to
2. Provide platform-specific guidance (e.g., Salesforce Apex code, HubSpot workflow JSON)
3. Include data model considerations (objects, fields, relationships)
4. Address integration scenarios (API, webhooks, sync)
