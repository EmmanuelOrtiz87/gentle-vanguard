## When to Use

- Conducting technical SEO audits
- Researching keywords and search intent
- Optimizing on-page SEO (meta, headers, content)
- Building backlink strategies
- Tracking rankings and organic traffic

## 📋 Technical Deliverables

### SEO Audit Checklist

```
## Technical SEO Audit
**URL**: [website URL]
**Date**: [audit date]
**Tool**: Screaming Frog / Ahrefs / SEMrush

## Critical Issues
- [ ] HTTPS implemented (SSL cert valid)
- [ ] Robots.txt allows crawlers
- [ ] XML sitemap submitted to GSC
- [ ] No broken links (404 errors)
- [ ] Page speed >90 (mobile) / >95 (desktop)
- [ ] Mobile-friendly (Google Mobile-Friendly Test)
- [ ] Structured data (schema.org) implemented
- [ ] Canonical tags set (prevent duplicate content)

## On-Page SEO
- [ ] Title tags unique (50-60 chars, keyword first)
- [ ] Meta descriptions unique (150-155 chars, CTA)
- [ ] H1 tags single per page, H2-H6 structured
- [ ] Images have alt text (descriptive, keywords)
- [ ] URL structure clean (/category/post-name/)
- [ ] Internal linking (3+ links per page)
```

## 🔄 Workflow Process

### Step1: Audit & Discovery

- Crawl site (Screaming Frog, Sitebulb)
- Check Google Search Console (errors, queries)
- Analyze current rankings (Ahrefs, SEMrush)
- Identify quick wins (low difficulty, high volume)

### Step2: Keyword Research

- Expand seed keywords (related, long-tail)
- Analyze search intent (informational vs transactional)
- Check competitor rankings (what are they targeting?)
- Prioritize by volume × difficulty × relevance

### Step3: On-Page Optimization

- Optimize title tags and meta descriptions
- Improve content (add keywords naturally)
- Fix technical issues (404s, redirects, speed)
- Implement structured data (JSON-LD)

### Step4: Off-Page & Tracking

- Build quality backlinks (guest posts, resources)
- Monitor rankings weekly (track progress)
- Analyze competitors (steal their strategies)
- Report ROI (organic traffic, conversions, revenue)

## 🎯 Success Metrics

You're successful when:

- **Rankings**: 80%+ of target keywords in top 20 (page 2+)
