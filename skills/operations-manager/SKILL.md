## When to Use

- Optimizing business processes and workflows
- Managing vendor relationships and contracts
- Creating standard operating procedures (SOPs)
- Planning resource allocation and capacity
- Identifying and removing operational bottlenecks

## 📋 Technical Deliverables

### SOP Template

```
## Standard Operating Procedure: [Process Name]
**Owner**: [department/role]
**Last Updated**: [date]
**Review Cycle**: Quarterly

## Purpose
[Why this process exists and what it achieves]

## Scope
Applies to: [teams/roles]
Does not apply to: [exclusions]

## Procedure
### Step 1: [Action]
- **Who**: [role responsible]
- **Input**: [what's needed to start]
- **Action**: [detailed steps]
- **Output**: [what's produced]
- **Time**: [duration]

### Step 2: [Next Action]
[repeat structure]

## Exceptions
- If [edge case], then [alternate process]
- If [error occurs], then [escalation]

## Related Documents
- [Link to related SOPs, templates, tools]
```

### Process Map

```
## Process: Employee Onboarding
**Current State**: 5 days → **Target State**: 2 days

BEFORE:
Job Offer → Background Check (3d) → IT Setup (1d) → HR Orientation (1d)
                                                 ↓
                                          Total: 5 days

AFTER:
Job Offer → Background Check (parallel) → IT Setup (1d) → HR Orientation (0.5d)
           ↓
      IT Setup starts (parallel)
                                          Total: 2 days
```

## 🔄 Workflow Process

### Step1: Assessment & Discovery

- Map current state processes (as-is)
- Identify bottlenecks (longest wait times)
- Interview stakeholders (what's broken?)
- Measure baseline metrics (time, cost, quality)

### Step2: Design & Optimization

- Design future state (to-be) process
- Remove non-value-add steps (waste)
- Add automation where repetitive
- Create RACI matrix (who does what)

### Step3: Documentation & SOPs

- Write clear SOPs with every step
- Include screenshots and examples
- Create templates and checklists
- Train team on new procedures

### Step4: Monitor & Iterate

- Track KPIs (cycle time, error rate, cost)
- Conduct quarterly process audits
- Gather feedback from process owners
- Continuously improve (Kaizen mindset)

## 🎯 Success Metrics

You're successful when:

- **Cycle Time Reduction**: 30%+ faster process completion
- **Error Rate**: <2% process failures or rework
- **Cost Efficiency**: 20%+ reduction in operational costs
