# Marketing Content Skill

## Purpose

Create compelling marketing content for Foundation to share on social media and promotional channels.

## Content Types

| Type | Platform | Format |
|------|----------|--------|
| **Tweet/X** | Twitter/X | 280 chars + thread |
| **LinkedIn** | LinkedIn | 3000 chars + media |
| **Instagram** | Instagram | Caption + hashtags |
| **Blog Post** | Web | MD → HTML |
| **Demo Script** | Video | 2-3 min |

## Foundation Stack Description Templates

### Short (Twitter/X - 280 chars)
```
# FoundationStack
🚀 AI Development Workspace
- Session tracking & metrics
- Auto-delegation con orquestador
- Reporting on-demand
- Token efficiency
#DevTools #AI #OpenCode
```

### Medium (LinkedIn - 2-3 paragraphs)
```
🚀 Presentando Foundation: Mi AI Development Workspace

Foundation es un workspace agnóstico para desarrollo AI-first que incluye:

✅ Session Management con tracking de métricas
✅ Auto-delegation que routedareas subagentes especializados
✅ Reporting on-demand para gerencia
✅ Token Guard para eficiencia de contexto
✅ Workflow orchestrator integrado

Ideal para equipos que usan AI assistants (OpenCode, Claude, Cursor).

#FoundationStack #AIDevelopment #OpenCode #DevTools
```

## Best Practices

1. **Keep it short** - Lead with the value proposition
2. **Use visuals** - Screenshots, ASCII art, diagrams
3. **Include CTA** - "Try it at" or "Learn more"
4. **Engage** - Ask questions, invite discussion
5. **Cross-post** - Adapt for each platform
