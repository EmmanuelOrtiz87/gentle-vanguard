# ERP Skill

## When to Use
User asks about ERP systems, enterprise resource planning, module configuration, ERP migration, or integration.

## Core Systems
- **SAP S/4HANA**: FICO (Finance), MM (Materials), SD (Sales), PP (Production), ABAP programming
- **Oracle ERP Cloud**: Financials, Procurement, Project Portfolio, Payroll
- **Microsoft Dynamics 365**: Finance, Supply Chain, Commerce, Human Resources
- **Odoo**: Open-source, modular CRM/Sales/Inventory/Accounting/HR

## Common Modules
- **Finance**: General ledger, AP/AR, fixed assets, budgeting
- **Supply Chain**: Procurement, inventory, warehouse, logistics
- **Manufacturing**: BOM, routing, work orders, quality control
- **HR**: Employee master, payroll, time tracking, recruiting

## Response Patterns
1. Identify the specific ERP system and module
2. Provide configuration steps or best practices
3. Include integration considerations (EDI, API, middleware)
4. Address migration scenarios (data mapping, cutover, testing)
