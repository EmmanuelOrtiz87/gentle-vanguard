## Quick Reference

### Project Types

- `service` - REST APIs, workers, daemons
- `cli` - Command-line tools
- `library` - Reusable packages
- `frontend` - Web applications
- `fullstack` - Frontend + Backend
- `microservices` - Distributed systems
