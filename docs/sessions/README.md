# 🎮 Sessions#

<p align="center">
  <b>Session lifecycle artifacts and telemetry</b>
</p>

---

## 📋 Purpose#

| Goal              | Description                                                         |
| ----------------- | ------------------------------------------------------------------- |
| **🎯 Tracking**   | Track session starts, context packs, and closure outputs            |
| **🔄 Continuity** | Preserve continuity between work blocks and day-end handoffs        |
| **📊 Telemetry**  | Support orchestrator and operator workflows with concrete artifacts |

---

## 📂 Directory Structure#

```
docs/sessions/
├── 📄 README.md                           # This file - sessions hub
├── 🎮 YYYY-MM-DD-HHmmss-session-start.md    # Session brief snapshot
├── 📦 YYYY-MM-DD-HHmmss-context-pack.md    # Compact context handoff summary
├── 📋 YYYY-MM-DD-HHmmss-delivery-closure.md # Operational closure output
├── 📊 closure-report-YYYY-MM-DD.md        # Day-end consolidated closure report
├── 📈 metrics/                             # Session telemetry
│   └── context-usage.csv                   # Context usage metrics
└── 🗂️ .local-archive/                       # Archived older sessions
```

---

## 🚀 Quick Start#

### Start New Session#

```powershell
.\scripts\utilities\wf.ps1 start-session
# Output: docs/sessions/YYYY-MM-DD-HHmmss-session-start.md
```

### Generate Context Pack#

```powershell
.\scripts\wf.ps1 context-pack
# Output: docs/sessions/YYYY-MM-DD-HHmmss-context-pack.md
```

Captures: branch, recent commits, changed files, platform health.

### Compact Context#

```powershell
.\scripts\wf.ps1 compact-start "<objective>"
# Reads: latest context-pack (by timestamp)
# Logs: docs/sessions/metrics/context-usage.csv
```

### End Session#

```powershell
.\scripts\utilities\wf.ps1 end-session
.\scripts\utilities\wf.ps1 day-end-closure
```

## 📊 Session Artifacts#

| Artifact                | Description                              | Generated by         |
| ----------------------- | ---------------------------------------- | -------------------- |
| **🎮 Session Start**    | Brief snapshot of objectives and context | `wf start-session`   |
| **📦 Context Pack**     | Compact handoff summary for continuity   | `wf context-pack`    |
| **📋 Delivery Closure** | Operational closure output               | `wf end-session`     |
| **📊 Closure Report**   | Day-end consolidated report              | `wf day-end-closure` |
| **📈 Metrics**          | Telemetry: events, packs, efficiency     | `wf context-metrics` |

## 🔍 Session Workflow#

### Step 1: Start Session#

```powershell
.\scripts\utilities\wf.ps1 start-session
# Auto-generates: docs/sessions/YYYY-MM-DD-HHmmss-session-start.md
```

### Step 2: Work & Track#

```powershell
# Daily commands
foundation verify  # Check all 14 quality gates
foundation version # Show stack version + skills count
wf dashboard       # Open HTML metrics dashboard
```

### Step 3: Compact Context (Optional)#

```powershell
.\scripts\wf.ps1 compact-start "Complete feature X"
# Loads latest context pack and records telemetry
```

### Step 4: End Session#

```powershell
.\scripts\utilities\wf.ps1 end-session
# Generates: docs/sessions/YYYY-MM-DD-HHmmss-delivery-closure.md
```

### Step 5: Day-End Closure#

```powershell
.\scripts\utilities\wf.ps1 day-end-closure
# Generates: docs/sessions/closure-report-YYYY-MM-DD.md
```

## 🚀 Quick Commands Reference#
