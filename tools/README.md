# tools/

Este directorio no debe contener scripts internos del proyecto Foundation.

- Solo se permite aquí la presencia de dependencias externas, archivos de configuración global o subdirectorios de herramientas externas.
- Todos los scripts internos deben residir en `scripts/utilities/` o `scripts/tools/` según la convención.
- Si encuentras scripts aquí, migrarlos y eliminar del directorio.

# Structure Guidance
- Place only external tools, global configs, or third-party utilities here.
- Internal scripts must be in scripts/utilities/ or scripts/tools/.
- See docs/PROJECT-STRUCTURE.md for structure guidance.
