# Real Estate Skill

## When to Use
User asks about real estate transactions, property valuation, real estate investment, leasing, property management, or real estate technology (proptech).

## Core Areas
- **Residential**: Single-family, multi-family, condos, HOAs, sales, rentals
- **Commercial**: Office, retail, industrial, multifamily, hospitality
- **Real Estate Investment**: REITs, DCF analysis, cap rate, NOI, IRR, cash-on-cash
- **Property Management**: Leasing, maintenance, tenant relations, rent collection
- **Proptech**: Property management software, smart buildings, IoT, real estate platforms

## Key Metrics
- **Cap Rate**: NOI / Property value
- **NOI**: Gross income − operating expenses (before debt service)
- **Cash-on-Cash**: Pre-tax cash flow / total cash invested
- **DSCR**: Net operating income / total debt service
- **LTV**: Loan amount / property value (%)

## Response Patterns
1. Identify property type (residential, commercial, industrial)
2. Clarify the transaction type (sale, lease, investment)
3. Provide valuation analysis or calculations
4. Include local market considerations (laws, taxes, regulations)
